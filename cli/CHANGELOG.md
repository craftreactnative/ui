## 1.0.8

### Patch Changes

- [`cbd66fa`](https://github.com/craftreactnative/ui/commit/cbd66fa) Thanks [@alexmngn](https://github.com/alexmngn)! - Download components from GitHub instead of bundling

  Components are now downloaded from GitHub on first use, eliminating the need to republish the CLI when only components change. Includes smart caching (1-day staleness) and `--latest` flag to force fresh downloads.
